Principais formatções

Negrito/ Destaque

Nessa frase temos um termo em negrito usando a tag B

Já nessa frase, temos um termo em negrito usando a tag STRONG (efeito semântico).

Itálico/Ênfase

Nesse parágrafo temos um termo em itálico usando a tag I.

Já nesse, temos um termo em itálico usando a tag EM (efeito semântico).

Texto marcado

Podemos criar um texto marcado usando a tag MARK.

Podemos criar um texto marcado usando a tag MARK, com estilo do parâmetro style.

Texto grande e texto pequeno

Podemos criar um texto grande usando a tag BIG, QUE JÁ ESTÁ OBSOLETA.
E também um texto pequeno usando a tag SMALL, essa não foi descontinuada, pois é usada para colocar as letrinhas miúdas em contratos e observações.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado com a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois com a tag INS.

Também um texto fica sublinhado com a tag U, mas essa não possui efeito semâtico e já não é indicada.

Texto sobrescrito

Para inserir coisas do tipo x20 + 3 com a tag SUP.

Texto subscrito

Para inserir coisas do tipo H2O com a tag SUB.
