Código-fonte e Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Essa formatação é por causa da tag CODE.

num = int(input('Digite um número: ')) if num % 2 == 0: print('O número é par.') else: print('O número é ímpar.') .Usando apenas a tag CODE, a identação do código não segue o padrão correto.

Para isso usamos a tag PRE junto com a tag CODE.

 
num = int(input('Digite um número: '))
if num % 2 == 0:
print('O número é par.')
else:
print('O número é ímpar.')
